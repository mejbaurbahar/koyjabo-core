## 🚀 Works 100% Offline!

**কই যাবো (Koy Jabo)** is Bangladesh's first transport app that works **completely offline**. After your first visit, enjoy all features without any internet connection!

### ✨ Offline Features

- ✅ **200+ Local Bus Routes** - Complete Dhaka bus database
- ✅ **Metro Rail Guide** - MRT Line 6 and upcoming lines
- ✅ **1000+ Intercity Routes** - Bus and train routes nationwide
- ✅ **AI Assistant** - Intelligent route planning (offline capable)
- ✅ **Fare Calculator** - Accurate pricing for all routes
- ✅ **Route Planner** - Multi-modal journey suggestions
- ✅ **Offline Maps** - Cached map tiles for navigation
- ✅ **Search History** - All your past searches
- ✅ **Favorites** - Quick access to saved routes
- ✅ **Travel Blog** - Cached guides and tips

### 🎯 How to Use Offline

1. **First Visit:** Open https://koyjabo.live while online (takes ~5 seconds)
2. **Automatic Caching:** App automatically downloads all data
3. **Go Offline:** Switch to airplane mode or disconnect internet
4. **Use Normally:** All features work perfectly!

## 🌟 Install as App

### Mobile (Android/iOS)
1. Visit https://koyjabo.live in Chrome/Safari
2. Tap "Add to Home Screen" when prompted
3. App installs like a native app
4. Launch from home screen - works offline!

### Desktop (Windows/Mac/Linux)
1. Open in Chrome/Edge
2. Click install icon in address bar
3. App opens in standalone window
4. Fully functional offline!
