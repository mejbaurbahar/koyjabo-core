### LOCAL vs INTERCITY Buses:

**Your Database = LOCAL DHAKA CITY BUSES** ✅

- Routes like Mirpur ↔ Motijheel, Gabtoli ↔ Gulistan
- Examples: Achiম Achim, Alif, Anabil, Balaka, BRTC City, etc.

**NOT in Database = INTERCITY BUSES** (Correct! ✅)

- Routes like Dhaka → Chittagong, Dhaka → Rangpur
- Examples: Nabil, Shohagh, Eagle, Green Line, Shyamoli
